# Docker Hands-on
Repository containing the Docker hands-on exercises completed during the Epigenomics course.

## Contents
- `Dockerfile`: Docker image definition for the R reproducibility exercise.
- `scripts/script.R`: Example R script executed inside the Docker container.
- `bind_test/`: Files generated during the bind mount exercise.
